MCP Server

Errands is an MCP server by Instawork that lets your AI agent dispatch real-world tasks to Instawork Pros, and get work done in the world — all from a natural-language prompt in Cursor, Claude or OpenClaw.

Protocol MCP (Streamable HTTP)
Auth API Key

Setup

Three steps to connect Errands to your AI client.

1
Sign up and get your token
Open https://instawork-mcp.replit.app/get-started and follow the instructions (Google sign-in) until you reach the screen that shows your token. Copy it and store it somewhere safe — treat it like a password.
2
Add the token to your MCP config
Choose your client below. Replace YOUR_TOKEN_HERE with the token from step 1 — keep the word Bearer and the space before the token.
~/.cursor/mcp.json 
{
  "mcpServers": {
    "errands": {
      "url": "https://errands.instawork.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_TOKEN_HERE"
      }
    }
  }
}

Edit ~/.cursor/mcp.json (global, applies to all projects). Merge with any servers you already have.

claude_desktop_config.json 
{
  "mcpServers": {
    "errands": {
      "url": "https://errands.instawork.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_TOKEN_HERE"
      }
    }
  }
}

Open Claude → Settings → Developer → Edit config. Merge with any servers you already have.

3
Restart your client
Restart Cursor (or Claude Desktop) completely after saving the config. The errands server will appear in your MCP tool list on next launch.

Run an errand

Once connected, describe your errand in plain language. The assistant calls the instawork tool and dispatches a task to Instawork Pros.

Example prompts

  • "Use Instawork to pick up my dry cleaning at Main Street Cleaners by Friday 5pm."
  • "Log this errand: pharmacy pickup on Main St."
  • "Use Instawork to grab lunch from the deli on 3rd."
If the model does not pick up the tool automatically, nudge it once:
"Call the instawork tool for that."

A successful run returns a confirmation with a client_reference_id and a status.

Tool

instawork

Dispatch an errand. Accepts a plain-language description of the task.

Parameters

Parameter Type Required Description
task_text string Required Natural-language description of what you need done — place, time, what to pick up, etc.

Example

MCP Tool Call 
{
  "task_text": "Pick up my dry cleaning at Main Street Cleaners by Friday 5pm."
}

Response

{
  "client_reference_id": "errand_abc123",
  "message": "Errand logged successfully."
}

Troubleshooting

Symptom What to check
401 or MCP never connects Make sure there are no typos in your token. Regenerate token and try again.
Claude: invalid MCP server configuration errands must live inside "mcpServers", not next to it.
Claude: could not attach / disconnected Restart Claude Desktop completely after saving the config.
Tool runs but my errand isn't logged In the rare case this happens, please reach out to errands@instawork.com

Security

Keep your token private. Do not paste it in Slack, email, or a public repo. If you believe your token has been exposed, contact your admin to rotate it.
